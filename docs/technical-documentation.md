# Technical Documentation
Technical Documentation
1. Project Structure

.The project follows a modular structure:
index.html = Main webpage
styles.css = Styling and responsive design
script.js = Interactivity and dynamic behavior
assets/images/ = Images used in the website
docs/ = Documentation files

2. Responsive Design

.Responsive layout is implemented using:
CSS Grid (Projects and About sections)
Flexbox (Navigation and Skills)
Media queries at max-width: 720px

.On smaller screens:
Navigation becomes collapsible
Project cards stack vertically
About section changes to single-column layout
